📖 Logic Cheatsheet

    
  

  

    

      
Decoder Rule

      
Expands $n$ binary address lines $\to 2^n$ distinct outputs. Exactly 1 output is HIGH at a time.

    

    

      
Encoder Rule

      
Compresses $2^n$ input lines $\to n$ output bits. Priority encoders resolve multiple active inputs by picking the highest index.

    

    

      
Key Expressions

      
1:2 Dec: D0 = E·A', D1 = E·A
2:4 Dec: Di = E·m_i
4:2 Enc: A1 = Y3+Y2, A0 = Y3+Y2'Y1
